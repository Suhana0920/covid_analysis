# COVID-19 Global Data Analysis

An interactive data analysis project analyzing global COVID-19 trends using Python and real-world data.

## Project Overview
This project analyzes COVID-19 data from 200+ countries covering the period 2020-2024. 
The analysis includes case trends, death rates, vaccination progress, and country comparisons.

## Charts and Visualizations
- Total COVID-19 Cases Over Time (India, USA, Brazil, UK)
- Top 10 Countries by Total Cases
- COVID-19 World Map
- Vaccination Rate vs Death Rate Comparison
- India Daily New Cases with 7-Day Rolling Average
- Correlation Heatmap of Key COVID Metrics

## Tech Stack
- Python
- Pandas, NumPy
- Matplotlib, Seaborn
- Plotly (Interactive Charts)
- Dataset: Our World in Data (OWID) — 429,435 rows

## Key Findings
- United States had the highest total cases (100M+)
- Brazil had the highest death rate (1.85%)
- United Kingdom started vaccination campaign first
- Countries with higher vaccination rates showed lower death rates
- India faced two major waves — Delta (April 2021) and Omicron (January 2022)
